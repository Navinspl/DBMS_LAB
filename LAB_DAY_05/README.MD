# DBMS Lab 5 – SQL JOIN Queries

## Overview

This project contains SQL queries for practicing **JOIN operations** in DBMS.
The queries combine data from multiple tables such as **Student, Course, Faculty, Enrollment, and Department**.

## Concepts Used

* INNER JOIN
* Table Aliases
* WHERE clause
* ORDER BY
* GROUP BY
* Aggregate Functions (COUNT, MAX)

## Tables Used

* Student
* Course
* Faculty
* Enrollment
* Department

## Tasks Covered

* Student and Enrollment relationships
* Course and Faculty mapping
* Student–Course information using multiple joins
* Department based filtering
* Sorting and analytical queries

## Purpose

To understand how **SQL JOIN operations** are used to retrieve related data from multiple tables in relational databases.
